# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 3.x     | :white_check_mark: |
| 2.3.x   | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a Vulnerability

You can report a potential vulnerability by creating a [new issue](https://github.com/marblejs/marble/issues).
